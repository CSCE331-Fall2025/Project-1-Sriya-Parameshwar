Achoo - An Allergy Detection Web App (2023):

This is an accessibility-focused web app using HTML, Java, and GitHub to allow users to discover what allergens they are most likely allergic to. It has a calendar-like front end, enabling users to simply select the dates on which they had allergic reactions. Then, the backend logs each date and web scrapes reputable allergy reporting websites to compute what the user is most likely allergic to based on their selection trends. Data is collected for at least 2 weeks before a report is generated. This was a project that taught me a lot about locally hosting a website, integrating external website data into the web app, and the full software development cycle.
